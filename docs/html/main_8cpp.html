Detailed Description

Main implementation file for the route planning system.

This file implements the command-line interface and core functionality for the route planning system, including driving, restricted driving, and combined driving-walking modes.

Finds optimal combined driving-walking route.

Parameters
gReference to the graph object
sourceStarting node ID
destinationTarget node ID
maxWalkTimeMaximum allowed walking time in minutes

Finds route that:

  1. Starts with driving
  2. Includes parking at a node
  3. Finishes with walking
  4. Respects max walking time

If no perfect route is found, provides up to two approximate solutions.
